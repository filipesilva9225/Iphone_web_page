iPhone 17 Pro Landing Page
Uma landing page moderna e responsiva desenvolvida para simular a apresentação do iPhone 17 Pro, focada em performance, design e experiência do utilizador.

🚀 Tecnologias Utilizadas
Este projeto foi construído com as seguintes ferramentas:

React 19: Biblioteca principal para construção da interface.

Vite: Ferramenta de build rápida para o desenvolvimento.

Tailwind CSS 4: Framework utilitário para estilização moderna e responsiva.

ESLint: Para garantir a qualidade e padronização do código.

📱 Funcionalidades
Seção Hero Impactante: Apresentação visual com gradientes e animações de entrada.

Destaques de Hardware: Detalhes técnicos sobre o chip A18 Pro e o sistema de câmaras de 48MP.

Seletor de Cores Dinâmico: Permite visualizar o dispositivo em diferentes acabamentos de Titânio (Azul, Natural e Laranja).

Tabelas de Especificações: Comparativo rápido entre os modelos Pro e Pro Max, incluindo tamanho de ecrã e duração de bateria.

Design Responsivo: Adaptado para diferentes tamanhos de ecrã.

🛠️ Como Executar o Projeto
Instalar as dependências:

Bash
npm install
Iniciar o servidor de desenvolvimento:

Bash
npm run dev
Gerar a versão de produção:

Bash
npm run build
📂 Estrutura de Pastas
src/components: Contém os componentes modulares da aplicação como NavBar, Hero, HighLights e Colors.

src/assets: Recursos estáticos como imagens e ícones do projeto.

public/img: Armazena as imagens dos modelos de iPhone e componentes internos utilizados na página.

Nota: Este projeto foi criado exclusivamente para fins educacionais.
